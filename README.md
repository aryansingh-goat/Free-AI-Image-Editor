# Free AI Image Editor 🎨

A lightweight, zero-dependency web tool that lets you edit images with **Airforce AI** models – no server, no build steps. Just open the HTML file in your browser.

### ✨ Features
- 🔗 Upload any image → automatically hosted via ImgBB
- 🤖 Choose from multiple AI models (Nanobanana, Flux, etc.)
- ✍️ Write a prompt describing the edit you want
- 🌊 Real-time streaming output from the AI
- 🔐 Your API keys stay local – nothing is sent anywhere but to the official APIs

### 🚀 How to Use
1. Get a free **Airforce API key** from [panel.api.airforce/dashboard](https://panel.api.airforce/dashboard)
2. Get a free **ImgBB API key** from [api.imgbb.com](https://api.imgbb.com/)
3. Open `Image Edit.html` in your browser (or host it on any static server)
4. Paste both keys into the form
5. Select a model, pick an image, write a prompt, and click **Edit Image**

### 📦 Files
- `Image Edit.html` – the entire app (HTML + CSS + JS in one file)
- `README.md` – you are here

### 🛡️ Privacy & Security
**Your keys are never stored, logged, or shared.** This tool runs entirely in your browser. The code is fully open – you can inspect every line. If you fork or redeploy, never commit real keys into the repository.
